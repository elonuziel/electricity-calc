גיבוי ענן — JSONBin.io

איך להגדיר גיבוי ענן?

  1. הרשם בחינם לאתר jsonbin.io.
  2. גש לעמוד ה-API Keys (בתפריט או בלוח הבקרה), צור Access Key חדש והעתק אותו.
  3. הדבק את המפתח בשדה "מפתח API (API Key)" למטה.

ליצירת גיבוי חדש: לאחר הזנת מפתח API, לחץ על שמור לענן. המערכת תיצור מזהה (Bin ID) חדש ותציג לך אותו. הצפין/שמור אותו!

לטעינת גיבוי קיים: הזן את מפתח ה-API ואת מזהה ה-Bin ID ששמרת בעבר, ולחץ על טען מענן.
